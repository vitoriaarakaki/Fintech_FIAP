# Fintech_FIAP

*FinTech Pro – Plataforma Web de Gestão Financeira Inteligente*

Visão Geral do Projeto

O FinTech Pro é uma plataforma web desenvolvida para auxiliar usuários no gerenciamento completo de suas finanças pessoais de forma simples, intuitiva e eficiente. O sistema foi projetado para oferecer uma visão consolidada da saúde financeira do usuário, permitindo o acompanhamento de receitas, despesas, investimentos, cartões e transações em um único ambiente digital.

Com uma interface moderna e responsiva, o sistema proporciona uma experiência amigável tanto em computadores quanto em dispositivos móveis, facilitando o controle financeiro diário e apoiando a tomada de decisões mais inteligentes sobre o uso do dinheiro.

Objetivo

Desenvolver uma solução web que permita aos usuários:

- Controlar receitas e despesas;
- Monitorar saldo financeiro em tempo real;
- Gerenciar cartões de crédito;
- Realizar transferências;
- Acompanhar investimentos;
- Visualizar relatórios financeiros;
- Organizar transações de forma prática e segura.

O projeto busca promover a educação financeira e auxiliar os usuários na construção de hábitos financeiros mais saudáveis.
